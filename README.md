## Op De Kant scraper
This simple app scrapes random quotes from [opdekant.nl](http://opdekant.nl/).

#### Install
Install the Op De Kant scraper with `npm` (or Yarn):
```
npm install
``` 
or 
```
yarn install
```

Setup a `.env` configuration file, which is based on `.env.dist`. Don't forget to set all values.

Run the server:
```
npm dev
```

#### Slack command
You can fetch a random quote using `/opdekant` as endpoint. Please note that this endpoint requires `POST` as method.
